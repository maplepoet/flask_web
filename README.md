==改进建议:==

1. 希望能够自行修改信息，因为现在几百例数据输进去难免有误。

2. 希望个人信息开设“房系”、“字辈”选项，便于搜索显示某一字辈多少人。（这个怎么样可以搞出来）

3. 个人信息单开一栏用作妻女信息填写，这样就不用再创建新的妻女信息页面，也可以避免相同的“×氏”难以录入的问题

4. 系统内置年号干支纪年法到公元纪年法的换算，导入相关数据，使得输入年号和干支可转换得出对应公元
